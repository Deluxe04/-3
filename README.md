## Описание проекта

Данная лабораторная работа представляет собой реализацию полиморфной системы контейнеров на языке C++ с использованием шаблонов и объектно-ориентированного программирования. Архитектура проекта разделена на два уровня:

- **Базовый слой (линейные структуры)** — АТД «Последовательность» (`Sequence<T>`), реализованный на базе `DynamicArray<T>` и `LinkedList<T>`. Обеспечивает унифицированный интерфейс для работы с одномерными данными, включая функциональные операции `Map`, `Where`, `Reduce`, поддержку итераторов и полиморфное управление памятью.
- **Целевой АТД (двумерная структура)** — «Квадратная матрица» (`SquareMatrix<T>`). Нелинейная алгебраическая структура, которая **инкапсулирует линейный контейнер** `DynamicArray<T>` и предоставляет над ним полный набор матричных операций: сложение, умножение (на скаляр и матрицу), вычисление нормы Фробениуса, элементарные преобразования строк/столбцов, а также решатель СЛАУ методом Гаусса.

### Основные возможности:

- **Динамический массив** (`DynamicArray<T>`) — массив с изменяемым размером
- **Связный список** (`LinkedList<T>`) — односвязный список
- **Последовательности** (`Sequence<T>`) — унифицированный интерфейс для работы с контейнерами
- **Mutable и Immutable** версии последовательностей
- **AdaptiveSequence** — адаптивная последовательность, автоматически выбирающая оптимальную реализацию
- **Builder** — паттерн для удобного создания последовательностей
- **Option<T>** — тип для безопасной обработки отсутствующих значений
- **Итераторы** — поддержка интерфейсов `IEnumerator<T>` и `IEnumerable<T>`
- **Map-Reduce** — базовые и дополнительные функциональные операции над последовательностями
- **Перегрузка операторов**
- **Квадратная матрица** - (`SquareMatrix<T>`) — квадратная матрица на базе DynamicArray<T> с полным набором алгебраических операций
- **Complex** — поддержка комплексных чисел со специализированным вводом/выводом и арифметикой
- **Решатель СЛАУ** — метод Гаусса для систем A·x = b с поддержкой всех типов коэффициентов
