## Описание проекта

Данная лабораторная работа представляет собой реализацию полиморфной системы контейнеров на языке C++ с использованием шаблонов и объектно-ориентированного программирования. Архитектура проекта разделена на два уровня:

- **Целевой АТД (двумерная структура)** — «Квадратная матрица» (`SquareMatrix<T>`). Нелинейная алгебраическая структура, которая **инкапсулирует линейный контейнер** `DynamicArray<T>` и предоставляет над ним полный набор матричных операций: сложение, умножение (на скаляр и матрицу), вычисление нормы Фробениуса, элементарные преобразования строк/столбцов, а также решатель СЛАУ методом Гаусса. И "Диагональная матрица"  (`DiagonalMatrix<T>`), предоставляет полный набор матричных операций, а также реализована решать системы линейных уравнений (СЛАУ) методом Гаусса.
- Был реализован Бенчмарк в (`benchmark.h`). Он создан, чтобы эмпирически доказать, что использование специализированной структуры (DiagonalMatrix) даёт выигрыш в производительности по сравнению с общим решением (SquareMatrix) при операции умножения на вектор. 

### Основные возможности:

- **Динамический массив** (`DynamicArray<T>`) — массив с изменяемым размером
- **Option<T>** — тип для безопасной обработки отсутствующих значений
- **Перегрузка операторов**
- **Квадратная матрица** - (`SquareMatrix<T>`) — квадратная матрица на базе DynamicArray<T> с полным набором алгебраических операций
- **Диагональная матрица** - (`DiagonalMatrix<T>`) — диагональная матрица на базе DynamicArray<T> с полным набором алгебраических операций
- **Complex** — поддержка комплексных чисел со специализированным вводом/выводом и арифметикой
- **Решение СЛАУ** — метод Гаусса для систем A·x = b с поддержкой всех типов коэффициентов
- **Бенчмарк** (`benchmark.h`)
- **Map-Reduce**
